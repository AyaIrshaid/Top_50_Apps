# Top_50_Apps


* Shows the top 50 free apps.
* Shows details of the selected app.


## Used:

API calls | Alamofire
Display data | UICollectionView
Custom views |  @IBDesignable/@IBInspectable
Image cash | NSCache
Web view | WKWebView
